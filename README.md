## Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm (v9+ recommended)

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>
cd soumya-jain-spark

# Install dependencies
npm install
```

### Development

```sh
# Start the development server
npm run dev
```
### Build for Production

```sh
npm run build
```

### Preview Production Build

```sh
npm run preview
```
